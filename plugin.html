chrome.debugger.remote

  1. This plugin requires Chrome v18 or greater
  2. Log in to Orion and go to the Plugins page.
  3. Paste the URL for this page and click Install. You should see a success message.
  4. This plugin provides the chrome.debugger.remote service. There are no UI features.

To enable trace debugging add "debug=trace" to the end of the plugin URL
